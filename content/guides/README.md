# Guides Content

Add new guide files here as Markdown.

Use this frontmatter shape:

```md
---
title: Your guide title
summary: One short summary sentence for cards and SEO.
publishedAt: 2026-03-30
updatedAt: 2026-03-30
tags:
  - DNS
  - Networking
eyebrow: Optional label
---
```

Files that start with `_` are ignored, so you can keep templates alongside the live content.
